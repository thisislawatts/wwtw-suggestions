Lighweight widget for handling user submissions.

== Description ==

User's can fill in the form contained within a widget. These submissions are then forwarded to the site's admin email address.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wwtw-suggestions` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go the `Appearance > Widgets`, where you will find a new Suggestions widget, just drag and drop it into Widget Areas
